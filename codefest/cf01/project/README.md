# Project — Simple Neural Network Accelerator

## Model
784 → 256 → 128 → 10

## Goal
Understand the compute and memory cost of a small neural network.

## Key idea
Most of the computation comes from multiply-accumulate operations in the fully connected layers.

## Why this project
This project connects AI concepts with hardware thinking by focusing on MACs, parameters, memory, and arithmetic intensity.

## Future direction
Map this small model into a hardware-friendly design and identify the main bottlenecks.
